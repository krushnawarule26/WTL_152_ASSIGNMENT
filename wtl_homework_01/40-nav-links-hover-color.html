Navigation Links with Hover Color

Instructions:

Hover your mouse over the navigation links above to see the color change effect.

The links start with light gray color and change to orange when you hover over them.

CSS Properties Used:

  • color: Changes the text color on hover
  • background-color: Changes the background color on hover
  • transition: Creates a smooth color change animation

Color Values:

  • Normal state: #ecf0f1 (light gray)
  • Hover state: #f39c12 (orange)
  • Hover background: #34495e (darker blue)

The transition property makes the color change smooth instead of instant.
