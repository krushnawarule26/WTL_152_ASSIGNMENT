Flexbox Three Equal Boxes

Box 1

This box takes up equal space as the others.

Box 2

All three boxes have the same width.

Box 3

They automatically adjust to fill the container.

CSS Properties Used:

  • display: flex: Makes the container a flexbox
  • justify-content: space-between: Distributes space between items
  • gap: 20px: Adds space between flex items
  • flex: 1: Makes each box take equal space

Key Benefits:

  • Boxes automatically adjust to equal widths
  • Responsive - they adapt to different screen sizes
  • Easy to add or remove boxes
  • No need to calculate percentages

Try resizing your browser window to see how the boxes maintain equal widths!
