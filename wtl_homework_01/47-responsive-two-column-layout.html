Responsive Two-Column Layout

Main Content

This is the main content area that takes up more space on larger screens.

On desktop screens, this appears as a two-column layout with the main content on the left and sidebar on the right.

On mobile devices (screens smaller than 768px), this becomes a single column layout.

Sidebar

This is the sidebar content that appears on the right side on larger screens.

On mobile devices, it appears below the main content.

CSS Properties Used:

  • display: flex: Creates the flexbox layout
  • flex: 2 and flex: 1: Controls column widths (2:1 ratio)
  • gap: 30px: Adds space between columns
  • flex-wrap: wrap: Allows items to wrap to new lines
  • @media query: Changes layout for small screens

Responsive Behavior:

  • Desktop (>768px): Two columns side by side
  • Mobile (â‰¤768px): Single column stacked
  • Breakpoint: 768px (common tablet/mobile breakpoint)

Key Features:

  • Automatic column sizing with flex properties
  • Smooth transition between layouts
  • Maintains readability on all screen sizes
  • No horizontal scrolling on mobile

Try resizing your browser window to see the responsive behavior!

When the window gets narrow, the layout will switch from two columns to a single column.
