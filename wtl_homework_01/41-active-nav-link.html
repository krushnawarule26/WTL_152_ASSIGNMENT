Active Navigation Link Highlight

Instructions:

Notice how the "About" link has a red background, indicating it's the current active page.

Hover over the links to see the hover effects. The active link has a different hover color.

CSS Classes Used:

  • .active: Styles the currently active page link
  • :hover: Styles links when you hover over them
  • .active:hover: Special hover style for the active link

Visual States:

  • Normal links: Light gray text, transparent background
  • Active link: White text, red background
  • Hover on normal: Orange text, dark blue background
  • Hover on active: White text, darker red background

In a real website, you would typically use JavaScript or server-side code to add the "active" class to the current page's link.
