Horizontal Navigation Bar

This is a horizontal navigation bar created using an unordered list.

The list items are displayed inline to create a horizontal layout.

Hover over the navigation links to see the hover effect.

Key Features:

  • Dark background color
  • White text for contrast
  • Hover effects on links
  • Proper spacing between items
  • Rounded corners on hover
