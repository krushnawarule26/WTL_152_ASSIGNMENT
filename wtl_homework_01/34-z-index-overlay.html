Z-Index Overlay Example

Background Box (z-index: 1)
Bottom Box (z-index: 0)
Overlay Box (z-index: 2)

Z-index controls the stacking order of positioned elements:

  • Higher z-index values appear on top
  • Lower z-index values appear behind
  • Elements with no z-index have a default value of 0
  • Only works with positioned elements (relative, absolute, fixed)

In this example:

  • Red overlay box (z-index: 2) appears on top
  • Blue background box (z-index: 1) appears in the middle
  • Green bottom box (z-index: 0) appears behind
