Button Focus Effect

Instructions: Click on the button below, then press the Tab key to move focus to the button. Notice how the border changes color when the button is focused.

The :focus pseudo-class applies styles when the element receives focus (like when tabbing to it or clicking it).

This is important for accessibility - users navigating with keyboard need to see which element is focused.

We used outline: none to remove the default browser focus outline and replaced it with our custom border.

Try tabbing between the buttons to see the focus effect!
