Border Radius Examples

This box has no border-radius (sharp corners).
This box has border-radius: 10px (slightly rounded corners).
This box has border-radius: 25px (very rounded corners).
This box has border-radius: 50px (completely rounded corners).

The border-radius property rounds the corners of an element.

Higher values create more rounded corners.

When border-radius equals half the element's width/height, it becomes a circle!
