Position Relative Offset Example

This is a normal box with no positioning offset.
This box has position: relative with top: 20px and left: 30px offset. Notice how it's moved down and to the right from its normal position.
This normal box shows where the offset box would normally be positioned.

Key points about position: relative:

  • The element stays in the normal document flow
  • It's offset from its normal position
  • Other elements don't move to fill the space
  • Useful for fine-tuning element positions

The offset box is moved 20px down (top: 20px) and 30px right (left: 30px) from where it would normally appear.
