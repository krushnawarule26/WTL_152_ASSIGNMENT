CSS Properties Used:

  • display: flex: Makes the container a flexbox
  • flex-direction: column: Stacks items vertically
  • gap: 15px: Adds space between flex items
  • max-width: 400px: Limits the container width

Key Benefits:

  • Boxes stack vertically in a clean column
  • Consistent spacing between items
  • Easy to add or remove boxes
  • Responsive design

Flex Direction Options:

  • row: Items in a row (default)
  • column: Items in a column
  • row-reverse: Items in reverse row order
  • column-reverse: Items in reverse column order

The flex-direction: column property makes the flex items stack vertically instead of horizontally.
